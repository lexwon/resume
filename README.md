# Alexander Wong Resume

Static GitHub Pages version of the resume.

## Local Preview

Open `index.html` directly in a browser, or run a simple static server:

```sh
npm start
```

Then visit `http://localhost:8000`.

To use a different port:

```sh
PORT=3000 npm start
```

## Publish With GitHub Pages

1. Create a GitHub repository named `resume`.
2. Push this folder to the repository's `main` branch.
3. In GitHub, open the repository settings.
4. Go to **Pages**.
5. Set **Source** to **Deploy from a branch**.
6. Set **Branch** to `main` and folder to `/root`.
7. Save.

The site will publish at:

```text
https://<github-username>.github.io/resume/
```

This repo intentionally does not use a custom domain.
